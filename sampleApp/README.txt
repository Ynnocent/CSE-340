Make sure to follow these steps to install Pnpm into your system. It is a package manager much like npm but it uses a virtual file system to run everything.

Steps:
1. Make sure Node is installed with the most recent verion. To check this "node --version"
2. Enter "corepack prepare pnpm@latest --activate"
3. Validate that you have the pnpm by typing "corepack pnpm --version"
